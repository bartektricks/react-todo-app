###To Do React App

This app uses localStorage to keep the data and also htmlSanitize to sanitize the entered text. 

####Allowed tags
- &lt;b&gt;
- &lt;strong&gt;
- &lt;i&gt;
- &lt;em&gt;

####Run locally
>yarn start
>npm start
####or
Check it out on [Heroku](https://mysterious-temple-73532.herokuapp.com/)
